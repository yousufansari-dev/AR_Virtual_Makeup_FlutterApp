💄 Virtual Makeup AI – Try Now Application
📌 Project Overview

Virtual Makeup AI – Try Now is an AI-powered hybrid application that allows users to virtually try cosmetic products in real time. The project is built using Flutter for mobile application development and HTML/CSS/JavaScript for the AI-based try-on interface.

The system integrates Firebase Authentication, AI face try-on technology, and a dedicated Admin Dashboard to manage the application efficiently.

🧠 Project Description

This application enables users to experience makeup products digitally before applying them physically. Using AI-based face detection and rendering, users can try different makeup products on their face in real time.

The application follows a hybrid architecture:

Flutter for the main mobile application

HTML, CSS, JavaScript for the AI Try Now module

WebView to connect Flutter with the web-based AI engine

Firebase Authentication for secure login & access control
